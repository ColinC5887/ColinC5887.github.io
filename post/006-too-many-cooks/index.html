Too Many Cooks!

A GMTK Jam 2023 Project.

 Posted on July 10, 2023

It’s a set of tracks for GMTK Jam 2023.

About the game and the tracks

These tracks are for the GMTK Game Jam 2023. The theme is roles reversed.

The design of our is like those kitchen runner games (overcooked for example) but the player is a fed-up customer who’s gotten impatient and decides to go make his own food, only the chef doesn’t want him there. The player has to run for various ingredients on a top-down view to collect a meal while avoiding and sabotaging chef(s).

As it’s only a 48-hour jam, there are only two tracks in the game:

  • Main menu theme
  • Gameplay theme

To match the gameplay, the tracks need to be:

  • A zany, cartoony vibe
  • Lots of chaos and hijinks
  • Goofy
  • Interesting

Download Looped music

These are prepared for Unity:

  • 01. Soulful Restaurant - Intro
  • 01. Soulful Restaurant - Loop
  • 02. Kitchen Chaos - Intro
  • 02. Kitchen Chaos - Loop

Design of music

01. Soulful Restaurant is for the main menu scene:

  • To form a contract to the gameplay, the main menu soundtrack is required to be calm, relaxing.
  • I try to make it typical jazz music.
  • Using typical jazz instrument (saxophone, trumpet, jazz piano)
  • No obvious difference between music sections, to make it calm and relaxing.

When the game starts, it will turn to 02. Kitchen Chaos, it’s the gameplay music:

  • Saxophone and trumpet are used to add some “funny” and “cartoony” feel to the sound.
  • Drumkit is used to add some energy to the music.
  • Several sections with obvious differences are added to the track. Some are used to make the music interesting while some are used for the player to relax.
