Dusky Woods

A ScoreSpace Jam

 Posted on September 25, 2023

Calm but adventure music in a dark forest.

About the game and the tracks

These tracks are for the ScoreSpace Jam #27. The theme is Cats + Guns or Micro-world. We picked the first one.

The game is simple, player just wonder in a forest and trying to defeat the crazy cats (with gun) and try to survive.

There are only two tracks in that game

  • Main menu theme
  • Gameplay theme

Download Looped music

These are prepared for Unity:

  • 01. Dusky Woods - Intro
  • 01. Dusky Woods - Loop
  • 02. Song of Serenity - Intro
  • 02. Song of Serenity - Loop

Design of music

01. Dusky Woods is for the main menu scene:

  • The drum makes it a bit exciting.
  • I try to make the music dark (and a little sad).
  • It’s not very long because the player doesn’t stay on the main page for long.

02. Song of Serenity is for the gameplay.

  • The player will fight these cats (with a gun). But according to the game designer, we don’t want it to be too exciting. So the music tries to be calm.
  • Using strings to make it a peaceful music. Also, the volume of drum kits is reduced.
